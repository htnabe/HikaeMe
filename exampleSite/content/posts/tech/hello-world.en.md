Markdown is a lightweight markup language for creating structured documents quickly. This article introduces the core syntax.

## 1. Headings

Use `#` for headings. More `#` means a deeper heading level.

## 2. Paragraphs and Line Breaks

Separate paragraphs with a blank line. To force a line break, add two trailing spaces or use `<br>`.

## 3. Emphasis

Use *italic* and **bold** text to emphasize words.

## 4. Lists

### Unordered List

- Item 1
- Item 2
  - Sub-item A
  - Sub-item B

### Ordered List

1. First item
2. Second item
3. Third item

## 5. Links

[Link text](https://www.example.com)

## 6. Images

![Alt text](/img/about-page-thumbnail.webp)

## 7. Quotes

> This is a quote.
> It can span multiple lines.

## 8. Code

Inline code uses backticks, and code blocks use triple backticks.

```python
def hello_world():
    print("Hello, World!")
```

## 9. Horizontal Rule

Use three or more `-`, `*`, or `_`.
