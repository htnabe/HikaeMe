Markdown es un lenguaje de marcado ligero para crear documentos estructurados rapidamente. Este articulo presenta la sintaxis esencial.

## 1. Encabezados

Usa `#` para los encabezados. Cuantos mas `#`, mas profundo es el nivel.

## 2. Parrafos y saltos de linea

Separa parrafos con una linea en blanco. Para forzar un salto, agrega dos espacios al final o usa `<br>`.

## 3. Enfasis

Usa *cursiva* y **negrita** para resaltar texto.

## 4. Listas

### Lista sin orden

- Elemento 1
- Elemento 2
  - Subelemento A
  - Subelemento B

### Lista ordenada

1. Primer elemento
2. Segundo elemento
3. Tercer elemento

## 5. Enlaces

[Texto del enlace](https://www.example.com)

## 6. Imagenes

## 7. Citas

> Esta es una cita.
> Puede ocupar varias lineas.

## 8. Codigo

El codigo en linea usa comillas invertidas y los bloques usan triple comilla invertida.

```python
def hello_world():
    print("Hello, World!")
```

## 9. Linea horizontal

Usa tres o mas `-`, `*` o `_`.

---

## Resumen

Markdown es flexible y practico. Con lo basico puedes crear documentos claros y bien estructurados.
