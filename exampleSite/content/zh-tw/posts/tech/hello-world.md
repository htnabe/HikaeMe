Markdown 是一種輕量標記語言，可快速建立結構化文件。本文介紹最常用的核心語法。

## 1. 標題

使用 `#` 表示標題。`#` 越多，層級越深。

## 2. 段落與換行

段落以空行分隔。若要強制換行，可在行尾加兩個空白或使用 `<br>`。

## 3. 強調

可使用 *斜體* 與 **粗體** 強調重點。

## 4. 清單

### 無序清單

- 項目 1
- 項目 2
  - 子項目 A
  - 子項目 B

### 有序清單

1. 第一項
2. 第二項
3. 第三項

## 5. 連結

[連結文字](https://www.example.com)

## 7. 引用

> 這是一段引用。
> 可以跨越多行。

## 8. 程式碼

行內程式碼使用反引號，程式碼區塊使用三個反引號。

```python
def hello_world():
    print("Hello, World!")
```

## 9. 分隔線

使用三個以上 `-`、`*` 或 `_`。

Markdown 靈活且實用。掌握基礎後，就能快速撰寫清楚且結構良好的內容。
