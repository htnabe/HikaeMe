Markdown est un langage de balisage leger qui permet de rediger rapidement des documents structures. Cet article presente la syntaxe principale.

## 1. Titres

Utilisez `#` pour les titres. Plus il y a de `#`, plus le niveau est profond.

## 2. Paragraphes et retours a la ligne

Separez les paragraphes avec une ligne vide. Pour forcer un retour, ajoutez deux espaces en fin de ligne ou utilisez `<br>`.

## 3. Mise en valeur

Utilisez *italique* et **gras** pour souligner des mots importants.

## 4. Listes

### Liste non ordonnee

- Element 1
- Element 2
  - Sous-element A
  - Sous-element B

### Liste ordonnee

1. Premier element
2. Deuxieme element
3. Troisieme element

## 5. Liens

[Texte du lien](https://www.example.com)

## 7. Citations

> Ceci est une citation.
> Elle peut s'etendre sur plusieurs lignes.

## 8. Code

Le code en ligne utilise des backticks, les blocs de code utilisent trois backticks.

```python
def hello_world():
    print("Hello, World!")
```

## 9. Ligne horizontale

Utilisez trois `-`, `*` ou `_` (ou plus).

## Resume

Markdown est souple et pratique. En maitrisant les bases, vous produirez vite des contenus clairs et structures.
